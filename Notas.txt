Para poder clonar todo el repositorio con todo y las ramas es necesario que primero hagamos el 
git clone
luego el git branch -av para conocer todas las ramas que contiene el repo
una ves que sepamos cuales hay debemos cambiarnos a todas estas ramas para activarlas




Pasos
1.- Creamos el proyecto de react 
npx create-react-app login

2.- Instalamos firebase de manera global
npm install -g firebase

3.- Nos logemamos en firebase
firebase login

4.- Reparamos los errores
npm audit fix --force

5.- Instalamos las dependencias
npm i -S firebase react-with-firebase-auth
