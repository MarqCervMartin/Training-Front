## Setup

1. Instalar nvm
2. Instalar node con nvm
  nvm install v14.17.1
  nvm use 14.17.1
3. Instalar npm con nvm
  nvm install-latest-npm

Instalar yarn
npm install --global yarn

Instalar React
npm install -g create-react-app

Instalar Visual Studio Code

Extensiones: Prettier, Eslint
